# DevOps-mini-project

## 1st week - DevOps Basics, Linux, Git & Bash

# PROJECT
Writing a bash system health check script, pushed to GitHub and automated through GitHub Actions

## FOCUS
Understanding the basics of DevOps, the mindset, linux CLI, workflows, and Bash scripting.

## TOOLS
-	Linux (Ubuntu)
-	Bash scripting
-	Git / GitHub
-	GitHub Actions

## Features
- ✅ CPU usage monitoring  
- ✅ Memory and disk space checks  
- ✅ Network connectivity tests  
- ✅ Configurable thresholds
